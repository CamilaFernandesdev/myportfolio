Sobre o Projeto

O Tableau Scraper Energético foi desenvolvido para resolver um desafio crítico do setor energético brasileiro: a coleta automatizada de dados de consumo energético de todo o país. Durante minha atuação no setor de Mercado Livre de Energia da Copel, identifiquei a necessidade de automatizar um processo manual extremamente trabalhoso e propenso a erros.

O sistema processa dados de consumo energético nacional de empresas brasileiras, gerando históricos mensais com até 24 milhões de linhas de dados. A solução integra informações de múltiplas fontes governamentais (CCEE, ONS, ANEEL) e aplica filtros específicos para atender às necessidades do Mercado Livre de Energia.

Desafios Técnicos Superados

  • Navegação automatizada em sites com estruturas complexas e iframes
  • Processamento eficiente de 24 milhões de registros por execução
  • Sistema robusto de retry e tratamento de falhas
  • Integração com múltiplas APIs e formatos de dados

Organizações & Fontes de Dados

O sistema integra dados de três principais organizações do setor elétrico brasileiro, cada uma fornecendo informações específicas e essenciais para análises do Mercado Livre de Energia.

CCEE

Câmara de Comercialização de Energia Elétrica

www.ccee.org.br
Dados Coletados:
  • Consumo energético por empresa
  • Histórico mensal de consumo
  • Análises de mercado livre
  • Preços e liquidações

ONS

Operador Nacional do Sistema Elétrico

www.ons.org.br
Dados Coletados:
  • Capacidade de geração
  • Dados de transmissão
  • Demanda do sistema
  • Planejamento operativo

ANEEL

Agência Nacional de Energia Elétrica

www.aneel.gov.br
Dados Coletados:
  • Tarifas regulamentadas
  • Indicadores setoriais
  • Regulamentações
  • Fiscalização
