Sobre o Projeto

O Dashboard ONS foi desenvolvido para a Copel (Companhia Paranaense de Energia) como uma solução inovadora para análise de dados do ONS (Operador Nacional do Sistema Elétrico). O projeto atende às necessidades específicas dos engenheiros do setor que necessitam dos relatórios do ONS para realizar análises e insights semanais.

Com este dashboard é possível visualizar os principais reservatórios do país, realizar comparativos entre anos e analisar diferentes fontes geradoras de energia, otimizando tempo e permitindo análises mais concisas e eficientes.

O que é o ONS?

O ONS (Operador Nacional do Sistema Elétrico) é uma entidade privada, sem fins lucrativos, responsável pela coordenação e controle da operação das instalações de geração e transmissão de energia elétrica no Sistema Interligado Nacional (SIN) do Brasil.
