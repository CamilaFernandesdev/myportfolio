Sobre o Projeto

O App E-commerce Mobile foi desenvolvido para a ShopTech Inc. com o objetivo de criar uma experiência de compra mobile excepcional. O projeto envolveu a criação de um aplicativo nativo para iOS e Android usando React Native, com foco em performance, usabilidade e conversão.

O aplicativo conta com funcionalidades avançadas como busca inteligente, recomendações personalizadas, sistema de avaliações, chat de suporte em tempo real e integração completa com sistemas de pagamento e logística.

Desafios Enfrentados

  • Integração com múltiplos gateways de pagamento
  • Otimização de performance para catálogo com +10k produtos
  • Sincronização offline e online em tempo real
  • Sistema de notificações push personalizadas
