Sobre o Projeto

O Leitor PAR(p) é uma ferramenta especializada desenvolvida em Python para processar o arquivo de saída PAR(p) gerado pelo NEWAVE (Acervo CCEEE). Este arquivo contém modelos estocásticos de geração de energias sintéticas utilizando a metodologia PAR(p) - Processo Auto-Regressivo Periódico de ordem p.

Este projeto é fundamental para profissionais do setor elétrico que trabalham com planejamento energético e análise de cenários estocásticos, permitindo a extração e interpretação eficiente dos dados do arquivo PAR(p) do sistema elétrico brasileiro.

O que é o NEWAVE?

O NEWAVE é um modelo computacional desenvolvido pelo CEPEL (Centro de Pesquisas de Energia Elétrica) em conjunto com o CCEEE (Comitê Coordenador de Estudos Energéticos) para o planejamento da operação energética do Sistema Interligado Nacional (SIN).

Metodologia PAR(p)

O PAR(p) (Processo Auto-Regressivo Periódico de ordem p) é uma metodologia estatística avançada utilizada para modelar a estocasticidade das energias naturais afluentes aos reservatórios do sistema elétrico.

Sazonalidade: Captura padrões sazonais das afluências
Correlação: Modela dependências temporais e espaciais
Estocasticidade: Incorpora incertezas hidrológicas
Séries Sintéticas: Gera cenários futuros possíveis
