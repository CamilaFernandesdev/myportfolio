Sobre o Projeto

O CryptoMarketApp é uma aplicação iOS moderna desenvolvida durante o Santander BootCamp que implementa uma plataforma completa para listagem e visualização de criptomoedas. O projeto segue rigorosamente os princípios da Clean Architecture (VIP) e utiliza as tecnologias mais avançadas do ecossistema iOS.

Com foco em performance e experiência do usuário, a aplicação oferece gráficos interativos, sistema de notificações, persistência local com Core Data e gerenciamento reativo de estado usando o Combine Framework.

Arquitetura e Padrões

Clean Architecture (VIP)

Separação clara entre Domain, Data e Presentation layers com dependency injection

Repository Pattern

Abstração de fontes de dados com implementações local e remota

MVVM Pattern

Separação de responsabilidades entre Views e ViewModels usando Combine

Dependency Injection

Container próprio para inversão de controle e testabilidade

Funcionalidades Implementadas

Interface SwiftUI

Interface moderna e responsiva construída com SwiftUI e suporte a orientações

Gráficos Interativos

SwiftUI Charts com múltiplos timeframes (1H, 24H, 7D, 30D) e gestos de toque

Core Data

Persistência local otimizada com histórico de preços e operações em background

Sistema de Notificações

Alertas de preço configuráveis e notificações de mudanças significativas

Busca Avançada

Barra de busca com debounce e busca local/remota de criptomoedas

Pull-to-Refresh

Atualização de dados com cache inteligente e tratamento de estados

Modelo de Dados Robusto

O projeto implementa um modelo de dados completo para criptomoedas com mais de 25 propriedades, incluindo métricas de mercado avançadas:

Propriedades Essenciais: Preço, volume, ranking, capitalização
Métricas de Mercado: Variações 1h, 24h, 7d, 30d, 1 ano
Conformidade Codable: Serialização JSON robusta
Tratamento de Nulos: Valores opcionais bem tratados
